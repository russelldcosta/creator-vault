Brevsky's Library

A downloadable game for Windows

Download or claimName your own price

ABOUT:

A short horror experience set in an old library, run by a strange figure, Mr. Brevsky. You just work there as a maintenance, but you can't help noticing a strange mystery lurking behind the walls and old, dusty shelves.


MORE INFO:

~20 minutes

In game font: "jsl-ancient" by Jeff Lee

Page font: "Merriweather" by Eben Sorkin

Models are created by me, texture are from AmbientCG, 

spider web textures are made by "PolyOne Studio" on sketchfab



CONTROLS:

Movement W, A, S, D
Sprint Shift
Crouch Ctrl
Interaction E
ActionLMB

Gameplay examples

If you run into any bugs or issues, feel free to reach out in the comments, my YouTube channel or by e-mail at vukkdobric@gmail.com

Thank you!

UPDATE:

Added a downloadable image map of all spider web locations to help players who had trouble finding them.


Game has 2 Major endings.
